# 🌿 Spero Architectural Solutions
### *Defining the Next Generation of Architectural Finishes*



---

## 🏛️ Project Overview
**Spero** is a premium digital experience built for a leading architectural solutions provider. The platform showcases high-end wall and floor finishing solutions, blending industrial superiority with sustainable innovation. 

The website is designed with a **"Luxury & Prestige"** aesthetic, utilizing high-contrast visuals, sophisticated typography, and smooth scroll-triggered animations to mirror the quality of the physical products.

---

## ✨ Key Features
- **💎 Immersive Visuals:** High-resolution galleries showcasing masterpiece projects.
- **🛠️ Technical Catalog:** Detailed product specifications including coverage, pot life, and packaging.
- **📖 Insights Blog:** A dedicated space for architectural trends and technical innovations.
- **🎨 Design System:** A built-in design language guide for brand consistency.
- **📱 Responsive Luxury:** A custom side-drawer navigation optimized for a premium mobile experience.
- **🌓 Adaptive Themes:** Seamless transition between "Deep Black" and "Warm Paper" modes.

---

## 🎨 Visual Identity
The project follows a strict design language to maintain its high-end positioning:

| Element | Specification |
| :--- | :--- |
| **Primary Color** | `#00A550` (Spero Green) |
| **Backgrounds** | `#000000` (Deep Black) & `#F8F7F4` (Warm Paper) |
| **Typography** | **Cormorant Garamond** (Serif) & **Inter** (Sans-serif) |
| **Animations** | Scroll-triggered fades and word reveals via Framer Motion |

---

## 🚀 Tech Stack
- **Frontend:** React 18 + TypeScript
- **Build Tool:** Vite
- **Styling:** Tailwind CSS (Utility-first luxury)
- **Animations:** Framer Motion
- **Icons:** Lucide React
- **Routing:** React Router DOM (HashRouter)

---

## 🛠️ Development Setup

1. **Clone & Install:**
   ```bash
   npm install
   ```

2. **Run Development Server:**
   ```bash
   npm run dev
   ```

3. **Build for Production:**
   ```bash
   npm run build
   ```

---

## 📂 Directory Structure
- `/pages`: Core views (Home, About, Portfolio, etc.)
- `/components`: Reusable UI elements (Navbar, Footer, FadeIn, etc.)
- `/types.ts`: Global TypeScript definitions.
- `/DesignSystem.tsx`: The living brand guide.
